An example of how to use React frontend to consume REST API endpoints written in Java Spring Boot.

## Getting Started

### Prerequisites

- Java 17+
- Gradle

### Build & Run

```bash
./gradlew bootRun
```

Open http://localhost:8080

### Build Project

```bash
./gradlew build
```

### Run Tests

```bash
./gradlew test
```
